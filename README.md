# copolio-portfolio

Bilingual (Ko/En) static portfolio site built with Next.js, Tailwind CSS, and shadcn/ui.

## Tech Stack

- **Framework**: Next.js 16 (App Router, Static Export)
- **Styling**: Tailwind CSS v4, shadcn/ui
- **Content**: MDX with YAML frontmatter (gray-matter)
- **Language**: TypeScript, React 19

## Getting Started

```bash
npm install
npm run dev       # Start dev server
npm run build     # Static export build
npm run lint      # ESLint
```
