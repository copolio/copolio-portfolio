## 프로젝트 개요

기존 상용 API 게이트웨이의 높은 비용과 제한된 커스터마이징 문제를 해결하기 위해 Go 언어 기반의 고성능 오픈소스 API 게이트웨이를 개발했습니다.

## 주요 업무

### 아키텍처 설계
- 플러그인 기반 미들웨어 체인 아키텍처를 설계하여 인증, 레이트 리미팅, 로깅 등 기능을 모듈화
- gRPC 기반 내부 통신 프로토콜을 구현하여 마이크로서비스 간 효율적인 데이터 교환 실현

### 성능 최적화
- 커넥션 풀링 및 keep-alive 최적화로 처리량 3배 향상
- sync.Pool을 활용한 메모리 할당 최소화로 GC 부하 감소
- Redis를 활용한 분산 레이트 리미팅으로 클러스터 환경에서의 일관성 보장

### 오픈소스 운영
- 기여 가이드라인 수립 및 코드 리뷰 프로세스 구축
- CI/CD 파이프라인 구성 (GitHub Actions 기반 자동 테스트 및 릴리즈)

## 성과

- 초당 10만 요청 처리 달성 (벤치마크 기준)
- GitHub 스타 500+ 달성, 외부 컨트리뷰터 10명 이상 참여
- 사내 프로덕션 환경 도입, 기존 상용 솔루션 대비 인프라 비용 40% 절감

## 기술적 도전과 해결

### 고가용성 보장
- **문제**: 단일 게이트웨이 인스턴스 장애 시 전체 트래픽 중단 위험
- **해결**: 헬스체크 기반 자동 페일오버 및 서킷 브레이커 패턴 적용으로 99.9% 가용성 달성

### 동적 라우팅 성능
- **문제**: 수천 개의 라우팅 규칙에서 매칭 성능 저하
- **해결**: Radix Tree 기반 라우팅 알고리즘 구현으로 O(log n) 복잡도 달성
